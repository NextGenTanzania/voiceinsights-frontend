# VoiceInsights SDK v208

Use the SDK contract to integrate external systems with VoiceInsights Orchestrator™.

Core workflow:
1. `createCampaign()`
2. `uploadContacts()`
3. `simulateCampaign()`
4. `launchCampaign()`
5. `getCampaignStatus()`
6. `getResponses()`
7. `getReports()`

The SDK is intentionally lightweight and REST-first so partners can also use plain HTTP and webhooks.
