# v210.1 — Executive Governance & Operations Control

## Purpose

This release formalizes the early-stage business governance model for VoiceInsights Africa.

The operating principle is simple:

**Operations Manager prepares. Founder appoints and authorizes. VoiceInsights Cloud executes.**

This avoids a heavy approval chain while preserving Founder control over strategic business actions.

---

## Founder / Executive Authority

Founder / Executive is the highest business authority in VoiceInsights Cloud.

Founder can:

- Approve Client
- Create Organization
- Invite Organization Admin
- Invite Operations Manager
- Replace Operations Manager
- Suspend Operations Manager
- Remove Operations Manager
- Transfer Operations Manager role
- View all Organizations
- Approve Project Activation
- Suspend Organization
- View Revenue Dashboard
- Platform Settings
- AI/System Settings
- Security & Audit
- Export all organization data
- Change Billing Plan
- Activate Enterprise License

---

## Operations Manager Authority

Operations Manager runs daily business operations.

Operations Manager can:

- Receive Demo Requests
- Contact Clients
- Schedule Meetings
- Create Client Record
- Upload Proposal
- Upload Contract
- Upload Invoice
- Submit for Approval
- Invite Organization Users after approval
- Manage Projects
- Assign M&E Officer
- Assign Data Analyst
- Assign Enumerators
- Monitor Campaigns

Operations Manager cannot:

- Approve Organization
- Delete Organization
- Suspend Organization
- Change Cloud Settings
- Change AI Settings
- Suspend Platform
- Invite Founder
- Invite another Operations Manager
- Replace Founder
- Remove Founder
- Change Executive Lock
- Create another Operations Manager
- Replace Operations Manager
- Remove Operations Manager
- Export all platform data

---

## Operations Manager Appointment Model

Operations Manager is not a permanent platform owner.

Operations Manager is a Founder-appointed operational role.

Founder can:

1. Invite the first Operations Manager.
2. Replace the current Operations Manager.
3. Suspend the current Operations Manager.
4. Remove the current Operations Manager.
5. Transfer Operations Manager role to another user.

Default governance: one primary Operations Manager at a time.

Future expansion may allow multiple Operations Managers, but only if Founder enables it.

## Invite Governance

Founder can invite:

- Operations Manager
- Organization Admin
- Super Admin, if used
- Internal Staff

Operations Manager can invite:

- Organization Admin
- M&E Officer
- Data Analyst
- Enumerator

Organization Admin can invite inside their organization only:

- M&E Officer
- Data Analyst
- Enumerator

Operations Manager cannot invite another Operations Manager.

## Governance Summary

This is a light governance model for the first commercial phase of VoiceInsights Africa.

It ensures:

- Operations Manager can move fast.
- Founder retains final business control.
- VoiceInsights Cloud automates execution after Founder approval.
- Operations Manager can be replaced by Founder without disrupting clients or projects.
