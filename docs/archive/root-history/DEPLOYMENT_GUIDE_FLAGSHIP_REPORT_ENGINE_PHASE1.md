# Deployment — Flagship Report Engine™ Phase 1

No database migration is required for Phase 1. The release adds a pure report compiler, quality gate, protected API routes, tests and a new authoring workspace.

1. `cd backend`
2. `npm install`
3. `npm test`
4. `wrangler deploy`
5. Push the `site/` directory to the connected GitHub Pages repository.
6. Sign in and open `/app/flagship-report-engine.html`.

## Production acceptance

Use a real completed pilot report model and verify:

- source evidence pointers;
- sample design and statistical annex data;
- executive decisions with owners and timelines;
- publication gate result;
- audience profile output;
- accessibility and editorial review.
